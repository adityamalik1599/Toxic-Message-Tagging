# Toxic-Message-Tagging
Building a moderation interface for reviewing toxic player messages in a game. The moderator’s task is to assign a Toxicity Type and Impact to each message and review all processed reports in a separate view.

## Live Demo
https://toxicmessagedashboard.netlify.app


## Features

- Message Queue with all reports
- Tag/Edit messages using modal
- Multi-select Toxicity Type
- Single-select Impact (Low, Medium, High, Critical)
- Processed Reports view
- Tagged vs Untagged count


## Bonus Features

- Filters (Impact)
- Mobile responsive UI
- Color-coded Impact


## Tech Stack
- React Js
- JavaScript
- CSS
- HTML
